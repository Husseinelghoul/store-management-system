# Store Management System for CMPS277
### Tech

We use a number of open source projects to work properly:
* [MySQL]
* [Pug] 
* [Bootstrap]
* [node.js]
* [Express]
* [jQuery]

### Installation
requires [Node.js](https://nodejs.org/) to run.
Install the dependencies and devDependencies and start the server.

```sh
$ cd dillinger
$ npm install -d
$ nodemon
```
